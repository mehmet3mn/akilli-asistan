Key Features:

Semantic Analysis: Meaning-based categorization via text vectorization.

Instant Monitoring: Real-time tracking of the screenshot directory (watchdog).

Technologies Used: Python, watchdog, pytesseract, sentence-transformers (Vectorization), ChromaDB (Vector Database).
